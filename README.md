# Kokoru

Kokoru is a web apps to be used as part of dota 2 information provider.

# Technology Stack

mongodb as database

react + redux as frontend framework

node js + graphql as backend engine

karma as test runer

Note: React file is compiled in one file using webpack

# Preparation

```bash
npm install
webpack
```

# Run Instruction

```bash
yarn start
```

# Run Test

```bash
yarn test:setup
yarn test
```
